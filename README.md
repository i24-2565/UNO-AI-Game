# UNO AI Game (Minimax vs Expectimax)

## Description
This project implements a simplified UNO game using adversarial search algorithms.

## Players
- Player 1: Minimax (Defensive Strategy)
- Player 2: Expectimax (Offensive Strategy)
- Player 3: Manual / AI Simulation

## Features
- UNO rules (color/number matching)
- Skip card implementation
- AI decision making using search algorithms
- Depth = 3 game tree

## Algorithms Used
- Minimax (Defensive)
- Expectimax (Offensive)

## Evaluation Function
Score = 50 − 5(CAI) + 2(Copp) + 3(S)

## Files
- uno_ai.ipynb → main implementation
- images → game tree & output
